NAME
       nvme-intel-get-max-lba - Send NVMe Get-Feature Command, returns feature ID (0xC1/0xC2)

SYNOPSIS
       nvme intel get-max-lba <device> [-n | --native-lba]

DESCRIPTION
       Retrieves the NVMe get-feature (0xC1 / 0xC2 for native max lba).

       The <device> parameter is mandatory and may be either the NVMe character device (ex: /dev/nvme0), or a namespace block
       device (ex: /dev/nvme0n1).

       On success, the returned get-feature log may be returned as an unsigned integer. With the native flag [-n], the log may be
       returned as the native maximum lba.

OPTIONS
       -n, --native-lba
           Print the native maximum LBA

       -j, --json
           Dump the output in json format

       -r, --raw
           Dump the output in raw format

EXAMPLES
       •   Print the maximum LBA:

               # nvme intel get-max-lba /dev/nvme0

       •   Print the maximum LBA:

               # nvme intel get-max-lba /dev/nvme0

       •   Print the native maximum LBA:

               # nvme intel get-max-lba /dev/nvme0 -n

       •   Print in JSON format:

               # nvme intel get-max-lba /dev/nvme0 -j

       •   Print in Raw format:

               # nvme intel get-max-lba /dev/nvme0 -r
